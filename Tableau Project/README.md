## Overview

This project simulates a real-world business intelligence engagement analyzing the operational performance of a retail superstore.

The goal of the project was to transform transactional retail data into actionable insights that support profitability optimization, product strategy decisions, marketing allocation, and operational risk management.

Using Tableau and analytical modeling, I explored profit drivers, return patterns, geographic performance, and advertising opportunities across multiple business dimensions.
